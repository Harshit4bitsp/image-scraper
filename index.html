HTML Image Scraper

A desktop application for extracting and downloading images from HTML content

Download and Installation

Choose one of the following options to get started:

Option 1: Download Source Code

Download ZIP

Extract the ZIP file and follow the installation instructions below.

Option 2: Clone the Repository

git clone https://github.com/yourusername/html-image-scraper.git
cd html-image-scraper
pip install -r requirements.txt
python image_scraper.py

Features

  • User-friendly GUI - Easy-to-use interface for extracting images
  • Customizable Image Selection - Filter images by CSS class
  • Directory Selection - Choose where to save downloaded images
  • Real-time Logging - Monitor download progress as it happens
  • Automatic Retries - Built-in retry mechanism for failed downloads
  • Multi-threading - Responsive UI during download operations

How It Works

  1. Paste your HTML content into the application
  2. Specify an output directory for the images
  3. Optionally enter a CSS class to filter specific images
  4. Click "Download Images" to start the extraction process
  5. Monitor progress in the log area

The application parses HTML using BeautifulSoup, extracts image URLs, and downloads them to your specified directory.

Requirements

  • Python 3.6 or higher
  • tkinter (included in standard Python installation)
  • beautifulsoup4
  • requests

Usage Example

# Install dependencies
pip install -r requirements.txt

# Run the application
python image_scraper.py

# Then:
# 1. Paste your HTML with image tags
# 2. Set output directory
# 3. Specify image class (optional)
# 4. Click "Download Images"
